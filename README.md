# LNHO Systems Website

Einfache statische Webseite für GitHub Pages.

## Dateien

- `index.html` – Hauptseite
- `impressum.html` – Impressum-Vorlage
- `datenschutz.html` – Datenschutz-Vorlage
- `haftung.html` – Haftungsseite
- `styles.css` – Design und responsive Layout
- `script.js` – mobiles Menü und Jahreszahl
- `assets/LNHO Systems Logo.png` – Logo-Datei

## Logo austauschen

Die Webseite nutzt diese Datei:

```text
assets/LNHO Systems Logo.png
```

Wenn ein offizielles Logo vorhanden ist, einfach die vorhandene Datei mit exakt gleichem Namen ersetzen.

## Nutzung mit GitHub Pages

1. Neues GitHub-Repository erstellen, z. B. `lnho-systems`.
2. Alle Dateien und Ordner aus diesem Paket in das Repository hochladen.
3. In GitHub unter **Settings → Pages** gehen.
4. Als Quelle `Deploy from a branch` auswählen.
5. Branch `main` und Ordner `/root` auswählen.
6. Speichern.

Danach ist die Seite über GitHub Pages erreichbar.

## Wichtig

Die Seiten `impressum.html`, `datenschutz.html` und `haftung.html` enthalten Platzhalter und sollten vor Veröffentlichung rechtlich und inhaltlich geprüft werden.
